# repo-solus

## Adding the repo

`sudo eopkg add-repo Cantalupo https://solus.cantalupo.com.br/eopkg-index.xml.xz`
 
## Enabling the Repository

`sudo eopkg enable-repo Cantalupo`

## Install

```bash
sudo eopkg it package
```

`sudo eopkg disable-repo Cantalupo`
